# Bill of Materials

**BOMs** manages bill of materials definitions, showing which materials are required to produce each product through specific processes.

![Bill Of Materials Overview](/img/screenshots/boms-overview.png)

## Overview

- **List of all BOMs** linked to processes and products
- **Material requirements** for each production process
- **Quantities and relationships** between inputs and outputs
- **Primary and alternate** material options

## Top Bar Controls

### Actions
- **Clear Filters**: Reset all applied filters

## BOMs Table

### Columns
- **Process Code**: Unique process identifier
- **Process Name**: Full process description
- **SKU**: Output product code
- **SKU Name**: Output product description
- **SKU Type**: Product type (e.g., Work In Progress, Finished Good)

### Sortable Columns
All columns have sort icons allowing you to order by:
- Process Code
- Process Name
- SKU
- SKU Name
- SKU Type

### Row Actions
Each BOM row has a **detail icon** (right side) to view full BOM details.

![BOM Details](/img/screenshots/boms-details.png)

## BOM Details View

Click the detail icon to view complete bill of materials:

**Page Header:**
- Title: "BOM Details: [Process Code]"
- **Back to BOMs** button: Return to list

### Process Information
- **Process**: Full process name and description
- **ID**: Process code

### Output Product Information
- **SKU Name**: Product being produced
- **SKU Type**: Product type (e.g., WIP, Finished Good)

### Materials Section

**Filter Tabs:**
- **Show All**: Display all materials
- **Finished/WIP Only**: Filter to finished goods and work in progress materials

**Materials Table Columns:**
- **SKU ID**: Material code
- **SKU Name**: Material description
- **SKU Type**: Material type (Raw Material, Work In Progress, etc.)
- **Quantity**: Amount required
- **Position**: Sort order or sequence
- **Primary Alternate**: Indicates if this is the primary material or an alternate

**Features:**
- Sortable columns
- **Clear Filters** button
- **Set Primary** button: Designate a material as primary alternate
- Item count at bottom (e.g., "4 items")

**Example materials:**
- IG-101-51: White Corn Bags (USA) - Raw Material - Quantity: 0.7926507937 - Primary
- IG-102-51: White Corn Bags (Spain) - Raw Material - Quantity: 0.7926507937 - Set Primary button
- IG-121-53: Lime - Raw Material - Quantity: 0.01642857143 - Position: 2
- IG-122: Water - Raw Material - Quantity: 1.261984126984 - Position: 3

## Common Workflows

### Viewing All BOMs
1. Navigate to **BOMs**
2. Review table of all bills of materials
3. Sort by **Process Name** to group by production step
4. Or sort by **SKU** to see BOMs by output product

### Viewing BOM Details
1. Click **detail icon** on BOM row
2. **BOM Details** page opens
3. Review:
   - Process information
   - Output product (SKU Name, Type)
   - Materials section with all inputs
4. See quantities required for each material
5. Identify primary vs. alternate materials

### Understanding Material Requirements
1. In BOM Details, review **Materials** table
2. **Quantity** column shows how much of each material is needed
3. Quantities are typically ratios or per-unit amounts
4. **Position** indicates order or sequence
5. Use this for:
   - Production planning
   - Material procurement
   - Cost estimation

### Managing Primary and Alternate Materials
1. In Materials table, review **Primary Alternate** column
2. **Primary** indicates the default material
3. Other materials are alternates (substitutes)
4. Click **Set Primary** to change which material is primary
5. Useful for:
   - Supplier changes
   - Material availability
   - Cost optimization

### Filtering Materials
1. Use **Show All** to see all materials including alternates
2. Use **Finished/WIP Only** to filter to specific types
3. Helps focus on relevant materials

## Understanding BOMs

**BOM structure:**
- **Process**: The production step or operation
- **Output**: The product being made (SKU)
- **Inputs**: The materials required (Materials table)
- **Quantities**: How much of each input is needed

**Primary vs. Alternate:**
- **Primary**: Default material used in production
- **Alternate**: Substitute materials that can be used instead
- Allows flexibility in material sourcing and availability

**Quantity interpretation:**
- Quantities are typically ratios or per-unit amounts
- Example: 0.7926507937 kg of corn per kg of output
- System scales quantities based on work order size

## Tips

- **Sort by Process Code** to see BOMs in process order
- **Sort by SKU** to group BOMs by output product
- Review **Quantity** values to understand material consumption ratios
- Use **Primary Alternate** to manage material substitutions
- **Set Primary** when changing default materials
- **Position** helps organize materials in logical order
- Filter to **Finished/WIP Only** to focus on intermediate products
- Use BOM data for:
  - Material requirements planning (MRP)
  - Cost estimation and pricing
  - Inventory planning
  - Supplier management
