# Resources

**Resources** manages production equipment and lines, including availability patterns, changeover times, and organisational attributes.

![Resources Overview](/img/screenshots/resources-overview.png)

## Overview

- **List of all production resources** (machines, lines, equipment)
- **Resource configuration** including type and availability
- **Changeover durations** for setup times
- **Display order** for scheduling views
- **Bulk upload** capability for multiple resources

## Top Bar Controls

### Actions
- **Bulk Upload**: Import multiple resources from file
- **Add Resource**: Create new resource manually
- **Clear Filters**: Reset all applied filters

## Resources Table

### Columns
- **Name**: Resource name/identifier
- **Type**: Resource type (e.g., MACHINE)
- **Availability**: Availability pattern name (e.g., "Mon to Fri")
- **Changeover Duration**: Setup/changeover time between work orders
- **Display Order**: Numeric order for display in schedules

### Sortable Columns
All columns have sort icons allowing you to order by:
- Name
- Type
- Availability
- Changeover Duration
- Display Order

### Row Actions
Each resource row has a **detail icon** (right side) to view and edit resource details.

![Resource Details](/img/screenshots/resources-details.png)

## Resource Details View

Click the detail icon to view and edit resource configuration:

**Page Header:**
- Title: "Resource Details: [Resource Name]"
- **Back to Resources** button: Return to list

### Basic Information Section

**Fields:**
- **ID**: Resource identifier (display only)
- **Name**: Resource name (display only)

### Configuration Section

**Resource Type:**
- Dropdown selector (e.g., Machine)

**Display Order:**
- Numeric value for ordering in schedules

**Default Changeover Duration:**
- Time value with unit dropdown (Minutes)
- Setup time between work orders

**Availability:**
- Dropdown selector (e.g., "Mon to Fri")
- Links to availability pattern

**Save Changes** button: Apply modifications

## Add New Resource

Click **Add Resource** to create a new resource:

**Page Header:**
- Title: "Add New Resource"
- **Back** button: Return to resources list

**Fields:**
- **Resource ID**: Enter unique resource ID
- **Resource Name**: Enter resource name
- **Resource Type**: Dropdown (e.g., Machine)
- **Display Order**: Numeric value (e.g., 0)
- **Default Changeover Duration**: Time value with unit dropdown (Minutes)
- **Availability**: Dropdown (e.g., "Not Selected")
- **Category**: Dropdown (e.g., "No Category")
- **Location**: Dropdown (e.g., "No Location")
- **Warehouse**: Dropdown (e.g., "No Warehouse")

**Save Resource** button: Create the resource

## Bulk Upload Resources

Click **Bulk Upload** to import multiple resources:

**4-step wizard:**

### Step 1: File Upload

**Upload your current Data File**
- Drag and drop area
- Or click to select a file
- **Next** button to proceed

### Step 2: Select Data

**Select the data you would like to import**

**Select Sheet:**
- Dropdown to choose worksheet (e.g., "Resources")

**Preview:**
Table showing imported data:
- **ID**: Resource IDs
- **Name**: Resource names
- **Type**: Resource types (MACHINE)
- **Availability**: Availability patterns (Mon to Fri)
- **Changeover**: Changeover durations (0 mins, 15 mins, 60 mins)
- **Display Order**: Display order values

**Buttons:**
- **Prev**: Go back
- **Next**: Continue

### Step 3: Map Fields

**Field mapping options**

**Map Required Fields:**

**System Fields** → **Your Data Fields**
- **Resource Id**: Dropdown to select field
- **Name**: Dropdown to select field
- **Type**: Dropdown to select field

**Buttons:**
- **Prev**: Go back
- **Next**: Continue

### Step 4: Preview

**Preview**

Table showing resources to be imported:
- **Resource Id**: IDs (e.g., RES_001, RES_002, RES_003...)
- **Name**: Names (e.g., Fryer, Flavouring Drum 1, Flavouring Drum 2...)
- **Type**: Types (MACHINE)

**Buttons:**
- **Prev**: Go back to edit
- **Save Resources**: Import the resources

## Common Workflows

### Viewing All Resources
1. Navigate to **Resources**
2. Review table of all production resources
3. Sort by **Name** alphabetically
4. Or sort by **Display Order** to see schedule order

### Editing Resource Configuration
1. Click **detail icon** on resource row
2. **Resource Details** page opens
3. Modify:
   - Resource Type
   - Display Order
   - Default Changeover Duration
   - Availability pattern
4. Click **Save Changes**

### Adding a Single Resource
1. Click **Add Resource** button
2. Enter:
   - Resource ID (unique identifier)
   - Resource Name
   - Resource Type
   - Display Order
   - Default Changeover Duration
   - Availability pattern
   - Category, Location, Warehouse (optional)
3. Click **Save Resource**

### Bulk Uploading Resources
1. Prepare file with resource data (ID, Name, Type, Availability, Changeover, Display Order)
2. Click **Bulk Upload** button
3. **Step 1**: Upload file
4. **Step 2**: Select worksheet and preview data
5. **Step 3**: Map fields (Resource Id, Name, Type)
6. **Step 4**: Preview and click **Save Resources**

### Setting Changeover Times
1. Open resource details
2. Set **Default Changeover Duration**
3. Specify time in minutes
4. System adds this time between work orders on schedule
5. Useful for:
   - Cleaning time
   - Setup time
   - Tool changes
   - Product changeovers

### Assigning Availability Patterns
1. Open resource details
2. Select **Availability** dropdown
3. Choose availability pattern (e.g., "Mon to Fri")
4. Pattern defines when resource is operational
5. Schedule respects availability constraints

## Understanding Resource Configuration

**Resource Type:**
- Categorizes resources (e.g., Machine, Line, Equipment)
- Used for filtering and reporting

**Display Order:**
- Controls order in schedule views
- Lower numbers appear first
- Helps organize schedule visually

**Changeover Duration:**
- Time required between work orders
- Added automatically by scheduler
- Prevents back-to-back scheduling without setup time

**Availability:**
- Links to availability pattern
- Defines operational hours/days
- System only schedules during available times

**Category, Location, Warehouse:**
- Organizational attributes
- Help categorize and filter resources
- Optional fields

## Tips

- **Sort by Display Order** to see resources in schedule order
- Set **Changeover Duration** to realistic setup times
- Use **Availability patterns** to model shifts, maintenance windows, and downtime
- **Display Order** should reflect logical production flow
- **Bulk Upload** is efficient for initial setup or updates
- **Resource Type** helps organize and filter resources
- **Category/Location/Warehouse** useful for multi-site operations
- Keep **Resource IDs** consistent and meaningful
- Review changeover times periodically to ensure accuracy
