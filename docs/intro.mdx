# Welcome to Planufacture

**Planufacture** is an intelligent production planning and scheduling platform designed specifically for process manufacturers in the food & beverage industry.

## What is Planufacture?

Planufacture helps you:

- **Plan smarter** - Create optimized production schedules that balance demand, capacity, and constraints
- **Execute faster** - Manage work orders, materials, and resources in real-time
- **Adapt quickly** - Respond to disruptions with AI-driven rescheduling
- **Gain visibility** - Track performance with comprehensive analytics and reporting

## Quick Start

### 1. Access the Platform

Log in to your Planufacture instance:
- **Production**: [example.planufacture.tech](https://example.planufacture.tech)

### 2. Navigate the Interface

The main navigation includes:

- **Schedule** - View and manage your production timeline
- **Work Orders** - Create and track production orders
- **Materials** - Manage inventory and allocations
- **Resources** - Configure lines, equipment, and labor
- **Reports** - Access analytics and performance metrics

### 3. Core Workflows

#### Creating a Work Order
1. Navigate to **Work Orders**
2. Click **New Work Order**
3. Select your product (SKU)
4. Enter quantity and due date
5. Review material requirements
6. Submit for scheduling

#### Viewing the Schedule
1. Go to **Schedule**
2. Use the timeline view to see planned work orders
3. Drag and drop to adjust timing
4. Check for conflicts or constraints
5. Publish changes

## Key Concepts

### Work Orders
Production orders that specify what to make, how much, and when.

### SKUs (Products)
The items you manufacture, with associated BOMs and routing information.

### Resources
Production lines, equipment, and labor capacity.

### Materials
Raw materials, ingredients, and packaging components.

### Scheduling
The process of assigning work orders to resources over time.
