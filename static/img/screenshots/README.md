# Screenshots Folder

This folder contains screenshots for the Planufacture documentation.

## Current Status

**Placeholder images** have been created for Schedule and Inventory modules to allow the docs to build without errors.

These are simple gray boxes with text labels - **replace them with actual screenshots** from the Planufacture application.

## How to Replace Placeholders

1. Take a screenshot of the relevant UI in Planufacture
2. Save it with the exact same filename as the placeholder
3. The documentation will automatically use the new image

## Screenshot Guidelines

- **Format:** PNG (preferred) or JPG
- **Resolution:** 1920x1080 or similar (high quality)
- **Crop:** Show relevant UI, remove unnecessary browser chrome
- **Privacy:** Ensure no sensitive data is visible
- **Consistency:** Use same browser and zoom level for all screenshots
